- Figures: Figures of the plots for each mixture and different nozzle.
- Results: The python code gives three files back:
            1. Mp4 of the measured jet (we don't upload it in github because there are too many, they are too big to upload, it's also not necessary).
            2. Png file of the measured jet length in mm and in px over time in ms.
            3. Txt file of the measured lengths, this is used to make a plot for each mixture.
